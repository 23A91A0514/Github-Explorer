# 🚀 GitHub Followers Explorer

GitHub Followers Explorer is a modern, responsive Progressive Web App (PWA) built with **HTML**, **CSS**, and **JavaScript** that uses the **GitHub REST API** to explore GitHub users and their profiles.

The application allows users to search for any GitHub username, view profile details, browse followers and following lists, explore public repositories, and seamlessly navigate between user profiles. It features a clean, responsive interface with real-time API integration and can be installed as a Progressive Web App (PWA) for a native app-like experience on desktop and mobile devices.

## ✨ Features

* 🔍 Search any GitHub user by username
* 👤 View profile details, bio, location, and GitHub statistics
* 👥 Explore followers and following lists
* 🔄 Navigate between GitHub profiles with a single click
* 📂 View all public repositories
* 🔗 Open GitHub profiles and repositories directly
* 📱 Fully responsive design for desktop, tablet, and mobile
* 🚀 Installable as a Progressive Web App (PWA)
* ⚡ Fast and lightweight user experience

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* GitHub REST API
* Progressive Web App (PWA)

## 🎯 Purpose

This project was developed to strengthen frontend development skills by working with REST APIs, asynchronous JavaScript, responsive UI design, and Progressive Web App technologies while creating a practical tool for exploring GitHub profiles.
